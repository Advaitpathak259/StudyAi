# Lumina Learning AI

Lumina Learning AI is a Full-Stack MERN application that transforms static PDFs into interactive learning environments. Using Google Gemini 1.5, the app provides context-aware AI chat, automated quiz generation, and interactive flashcards.

## Features

-   **AI Chat**: Context-aware retrieval where Gemini processes ONLY the uploaded document's content.
-   **Automated Quizzes**: Generate 5 multiple-choice questions with explanations from any PDF.
-   **Flashcard System**: Animated flip-cards for modern active recall study methods.
-   **PDF Processing**: Automatic text extraction from PDF files.
-   **Analytics Dashboard**: Track documents and study sessions.
-   **Dark Mode**: Professional light/dark theme persistence.
-   **Secure Auth**: JWT-based authentication for user data protection.

## Prerequisites

-   Node.js (v18 or higher)
-   MongoDB (Running locally or MongoDB Atlas URI)
-   Google Gemini API Key (Get it from [Google AI Studio](https://aistudio.google.com/))
